# PowerDesigner 配置

## 表格面板中显示表字段的 Code 项

默认显示字段的 Name 项，没有显示 Code 项，不是很方便。  
菜单 Tools -> Display Preferences... -> Table -> Advanced... -> Columns -> List columns 右边的 Select 图标 -> 勾选上 Code 一项，调整一下显示顺序就好了。

## 表属性显示备注列

双击任意一张表，弹出表属性对话框，切到 Columns 选项卡，点击 Customize Columns and Filer 图标，勾选上 Comment 一项。

## 设置 Name 不自动等于 Code

Tools -> Gerneral Options... -> Dialog -> 去掉勾选 Name to Code mirroring
